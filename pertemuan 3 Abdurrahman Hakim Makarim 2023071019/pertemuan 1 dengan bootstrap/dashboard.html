Komputasi Berbasis Web

Komputasi web atau pemrograman web adalah ilmu yang mempelajari cara membuat dan mengembangkan situs web menggunakan HTML, CSS, dan Bootstrap. HTML digunakan untuk membangun struktur halaman, CSS untuk mengatur tampilan seperti warna dan tata letak, sedangkan Bootstrap mempermudah pembuatan desain responsif dengan komponen siap pakai.

Dalam mata kuliah ini, mahasiswa belajar membuat aplikasi web yang menarik dan responsif di berbagai perangkat. HTML membentuk dasar konten, CSS menata tampilannya, dan Bootstrap membantu mempercepat proses pengembangan dengan elemen desain modern yang siap digunakan.
